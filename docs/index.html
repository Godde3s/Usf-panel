🔑 توکن HuggingFace خود را وارد کنید

برای ساخت Space رایگان، به یک توکن HuggingFace با دسترسی Write نیاز دارید. ساخت توکن کاملاً رایگان و کمتر از ۳۰ ثانیه است.

  1. به صفحه HuggingFace Tokens بروید
  2. روی دکمه «New token» کلیک کنید
  3. نام توکن: Usf Deploy (یا هر نامی)
  4. نوع توکن: Write ← بسیار مهم!
  5. روی «Create token» کلیک کنید و توکن را کپی کنید
توکن با hf_ شروع می‌شود و حدود ۳۷ کاراکتر است.

نام Space و رمز عبور مدیر را مشخص کنید. این تنظیمات بعداً قابل تغییر خواهند بود.

❓ سوالات متداول

آیا واقعاً رایگان است؟

بله! HuggingFace Spaces یک لایه رایگان با 16GB RAM و 2 vCPU ارائه می‌دهد که برای استفاده شخصی کاملاً کافی است. هیچ کارت اعتباری نیاز نیست.

آیا توکن من امن است؟

بله. توکن شما فقط در مرورگر شما استفاده می‌شود و به هیچ سروری ارسال نمی‌شود. تمام ارتباطات مستقیماً بین مرورگر شما و API های HuggingFace انجام می‌شود. بعد از استقرار، توکن را از صفحه تنظیمات می‌توانید حذف کنید.

اگر Space از قبل وجود داشته باشد چه؟

اگر Space با همان نام از قبل وجود داشته باشد، فایل‌ها به‌روزرسانی می‌شوند و تنظیمات دوباره اعمال می‌گردند. داده‌های قبلی شما (لینک‌ها، ترافیک) حفظ می‌شوند.

خطای "سهمیه تکمیل شده" چه означает؟

HuggingFace در لایه رایگان فقط اجازه می‌دهد یک Space همزمان روشن باشد. اگر Space های دیگری دارید، از صفحه Spaces آن‌ها را pause کنید و دوباره تلاش کنید.

پنل بعد از مدتی خاموش می‌شود؟

HuggingFace Spaces رایگان هر ۴۸ ساعت یا در صورت بی‌فعالیتی restart می‌شود. اما Usf Panel از SQLite استفاده می‌کند، بنابراین تمام داده‌های شما (لینک‌ها، ترافیک، تنظیمات) حفظ می‌شوند. Space به‌صورت خودکار دوباره روشن می‌شود.

چگونه سرعت را بیشتر کنم؟

راهنمای کامل بهینه‌سازی سرعت در GitHub موجود است. نکات کلیدی: mux را در کلاینت فعال کنید، IPv6 را غیرفعال کنید، DNS را از طریق پروکسی هدایت کنید.

چگونه از نشت IP جلوگیری کنم؟

راهنمای کامل ضد نشت در GitHub موجود است. مهم‌ترین موارد: IPv6 را در کلاینت ببندید، WebRTC را در مرورگر غیرفعال کنید، DNS را از طریق پروکسی هدایت کنید.

کدام کلاینت‌ها پشتیبانی می‌شوند؟

تمام کلاینت‌های VLESS: V2RayN (ویندوز)، V2RayNG (اندروید)، Streisand (iOS)، Shadowrocket (iOS)، Foxray (مک)، Nekoray (لینوکس/ویندوز). صفحه status دکمه‌های deep-link برای ۶ کلاینت محبوب دارد.
